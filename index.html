Projects

DNA Gene Comparison System

A hybrid application combining a FastAPI Python backend with C++ core logic to perform efficient DNA sequence comparisons using suffix trees. The system features a user-friendly frontend for DNA record management, secure information retrieval, organ donation registration, and high-accuracy genetic similarity analysis. Data is managed via text files, and the project highlights the performance advantage of suffix tree algorithms for large DNA inputs.

View on GitHub

Object Recognition using CIFAR-10

Collaboratively developed and compared multiple machine learning and deep learning models—including CNN, SVM, MLP, and ensemble methods—for image classification on the CIFAR-10 dataset. The project highlights model performance differences, with a CNN achieving the highest accuracy (77.44%) by effectively capturing spatial features. Dimensionality reduction with PCA was applied to classical models to improve efficiency.

Live Demo: your-cifar10-demo-url.com

Signal Frequency Analysis Web Application

Developed a web application for advanced frequency analysis of uploaded signal data using MEMO-ESPRIT and FFT algorithms. The app processes CSV signal inputs, performs spectral analysis, and visualizes results with interactive plots.
